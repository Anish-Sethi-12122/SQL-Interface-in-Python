## Getting Started  
Install dependencies:  
Copy-paste the following command to your terminal and click enter --  
`pip install mysql-connector-python rich`  

---

## Run the application:  
Copy-paste the following command to your terminal and click enter --  
`python main.py`  

---

## How It Works  
Log in with your MySQL Password (change host, and user if not set to default "localhost" and "root") and boom, start working with SQL right away.
